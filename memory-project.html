Jeu de carte Mémoire

Réalisation d'un jeu de cartes mémoire sur le thème de la journée de la Femme.
Le jeu est composé de 9 cartes.
Le but du jeu est de reformer toutes les paires avant la fin du chronomètre.
Le design des cartes met à l'honneur les femmes, avec un style minimaliste.

Languagues : html5, CSS3, Javascript
Logiciels : Adobe Illustrator
